Tide Traceをはじめる

Tide Traceは、痛みと服薬の記録を自分で見返すためのアプリです。

記録はこのブラウザ内に保存されます。必要に応じてバックアップしてください。

      薬設定

      薬ボタンの表示名、よく使う量、単位、表示順を設定します。
      非表示にした薬は新規記録ボタンには出ませんが、過去記録は残ります。

      別の薬に変える場合は、名前を上書きせず、元の薬を非表示にして新しい薬を追加してください。非表示にしても過去記録は残り、再開時には表示に戻せます。

      単位が違う記録は、診察用サマリーで別に集計されます。

      登録済みの薬

      痛み状態設定

      痛み記録で選ぶ状態名と表示順を設定します。

      別の状態に変える場合は、名前を上書きせず、元の状態を非表示にして新しい状態を追加してください。名前を上書きすると、過去記録や診察用サマリーの状態別集計が分かりにくくなることがあります。非表示にしても過去記録は残り、再度表示に戻せます。
