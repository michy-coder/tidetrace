Tide Trace

Local-first pain and medication log

Tide Traceをはじめる

Tide Traceは、痛みや服薬の記録を自分で見返すためのアプリです。

記録はお使いのブラウザ内に保存されます。必要に応じて、バックアップを保存してください。

薬ボタン

管理したい薬を登録します。あとから管理画面で変更できます。使わない欄は空欄のままにしてください。

    痛み状態

    痛みが変わりやすい場面や姿勢を登録します。あとから管理画面で変更できます。使わない欄は空欄のままにしてください。

      薬設定

      薬ボタンの表示名、よく使う量、単位、表示順を設定できます。

      別の薬に変更する場合は、名前を上書きせず、元の薬を非表示にしてから、新しい薬を追加してください。非表示にした薬は新規記録ボタンに表示されませんが、過去の記録は残ります。再開するときは、表示に戻せます。

      単位が違う記録は、診察用サマリーで別に集計されます。

      痛み状態設定

      痛み記録で選ぶ状態名と、表示順を設定できます。

      別の状態に変更する場合は、名前を上書きせず、元の状態を非表示にしてから、新しい状態を追加してください。 名前を上書きすると、過去の記録や診察用サマリーの状態別集計が分かりにくくなることがあります。非表示にしても過去の記録は残り、再開するときは表示に戻せます。
